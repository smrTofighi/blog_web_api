# Blog Backend API

This is a simple RESTful API for a blogging system, developed using Node.js and Express. It allows for creating, reading, updating, and deleting blog posts and categories.

## Features
- RESTful API structure
- MongoDB connection using Mongoose
- Category and post system
- Error handling and standardized responses
- Ready for extending with authentication or user roles

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv

## Getting Started
```bash
git clone https://github.com/smrTofighi/blog_web_api.git
cd blog_web_api
npm install
npm run dev
